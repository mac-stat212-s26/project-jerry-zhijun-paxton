Analysis I — Within-Tier Inequality

Lorenz Curves with Bootstrapped Gini Coefficients

A Gini coefficient point estimate is not sufficient for inference: it has no standard error as typically presented. We bootstrap 999 resamples per size tier to construct 95% percentile confidence intervals, confirming whether the ordering across tiers is statistically reliable.

Code 
# ── Gini helper ─────────────────────────────────────────────────────────────────
gini <- function(x) {
  x <- sort(x[x > 0 & !is.na(x)])
  n <- length(x)
  if (n < 2L) return(NA_real_)
  2 * sum(seq_len(n) * x) / (n * sum(x)) - (n + 1L) / n
}

gini_stat <- function(data, i) gini(data[i])

set.seed(112)
gini_ci <- co |>
  group_by(size) |>
  group_modify(~ {
    b  <- boot(.x$capital_stock, gini_stat, R = 999)
    ci <- boot.ci(b, type = "perc", conf = 0.95)
    tibble(
      g     = b$t0,
      lower = ci$percent[4L],
      upper = ci$percent[5L],
      n     = nrow(.x)
    )
  }) |>
  ungroup()

gini_ci |>
  mutate(across(c(g, lower, upper), ~ round(.x, 4))) |>
  gt() |>
  tab_header(
    title    = md("**Bootstrapped Gini Coefficients by Firm Size**"),
    subtitle = "999 resamples · 95% percentile confidence interval"
  ) |>
  cols_label(
    size  = "Tier", g = "Gini",
    lower = "CI Lower", upper = "CI Upper", n = "N"
  ) |>
  fmt_number(columns = n, use_seps = TRUE, decimals = 0)

Theil T Decomposition: Within vs. Between Groups

The Lorenz curve compares tiers but cannot partition total inequality into within-tier and between-tier components. The Theil T index (GE(1)) supports an exact additive decomposition:

\[T = T_{\text{between}} + T_{\text{within}} = \sum_g \frac{n_g}{N}\frac{\mu_g}{\mu}\ln\frac{\mu_g}{\mu} \;+\; \sum_g \frac{n_g}{N}\frac{\mu_g}{\mu} T_g\] where \(T_g\) is the within-group Theil T for group \(g\), \(n_g\) is the group count, \(\mu_g\) is the group mean capital, and \(\mu\) is the overall mean.

Code 
theil_t <- function(x) {
  x  <- x[x > 0 & !is.na(x)]
  mu <- mean(x)
  mean((x / mu) * log(x / mu))
}

decomp <- co |>
  group_by(size) |>
  summarise(
    n_g = n(),
    mu_g = mean(capital_stock),
    T_g  = theil_t(capital_stock),
    .groups = "drop"
  ) |>
  mutate(
    N             = sum(n_g),
    mu            = sum(n_g * mu_g) / N,
    w_g           = (n_g / N) * (mu_g / mu),
    T_between_g   = w_g * log(mu_g / mu),
    T_within_g    = w_g * T_g
  )

T_between <- sum(decomp$T_between_g)
T_within  <- sum(decomp$T_within_g)
T_total   <- T_between + T_within

# Panel A: total within vs between
summary_bar <- tibble(
  component = c("Between tiers", "Within tiers"),
  value     = c(T_between, T_within),
  pct       = c(T_between / T_total, T_within / T_total)
)

p_summary <- ggplot(summary_bar, aes(x = component, y = pct, fill = component)) +
  geom_col(width = 0.55, alpha = 0.9) +
  geom_text(aes(label = percent(pct, accuracy = 0.1)),
            vjust = -0.5, fontface = "bold", size = 4.2) +
  scale_y_continuous(labels = percent_format(), limits = c(0, 1),
                     name = "Share of total Theil T") +
  scale_fill_manual(
    values = c("Between tiers" = "#264653", "Within tiers" = "#e76f51"),
    guide  = "none"
  ) +
  labs(title    = "Total decomposition",
       subtitle = glue("Total Theil T = {round(T_total, 3)}"),
       x = NULL) +
  theme_minimal(base_size = 12) +
  theme(plot.title = element_text(face = "bold", colour = "#264653"),
        panel.grid.major.x = element_blank(),
        panel.grid.minor   = element_blank())

# Panel B: within-tier contributions broken down by size
p_within <- decomp |>
  mutate(pct_of_total = T_within_g / T_total) |>
  ggplot(aes(x = fct_rev(size), y = pct_of_total, fill = size)) +
  geom_col(width = 0.55, alpha = 0.9) +

  scale_fill_manual(values = pal_size, guide = "none") +
  labs(
    title    = "Within-tier breakdown by size",
    subtitle = "Each bar = one tier's contribution to the within-group component",
    x = NULL
  ) +
  theme_minimal(base_size = 12) +
  theme(plot.title = element_text(face = "bold", colour = "#264653"),
        panel.grid.major.x = element_blank(),
        panel.grid.minor   = element_blank())

p_summary + p_within +
  plot_annotation(
    title   = "Most of Brazil's capital inequality is *within* size tiers, not between them",
    subtitle = glue(
      "Theil T decomposition. Within-tier component = {percent(T_within / T_total, 0.1)} ",
      "of total inequality — the largest driver is inequality inside micro-enterprises."
    ),
    caption = caption_base,
    theme   = theme(
      plot.title    = element_markdown(face = "bold", size = 14, colour = "#264653"),
      plot.subtitle = element_text(colour = "grey40", size = 10.5, lineheight = 1.3),
      plot.caption  = element_text(colour = "grey50", size = 8)
    )
  )

The decomposition is the key structural result of this section. If most inequality were between tiers, then adjusting the revenue classification cutoffs could reduce aggregate inequality. But if the dominant component is within tiers, revenue-threshold reform achieves little: the same cutoff that separates micro from small cannot distinguish capital-poor subsistence firms from well-endowed holding-company subsidiaries within the micro category. The within-tier dominance directly motivates the governance analysis that follows.

The bootstrapped confidence intervals confirm that the Gini point estimates are precise: the
non-overlapping 95% CIs establish that all three tiers differ significantly in their degree of
internal inequality. Micro-enterprises (Gini > 0.60) are substantially more internally
unequal than either small or medium/large firms.

### Theil T Decomposition: Within vs. Between Groups

The Lorenz curve compares tiers but cannot partition total inequality into *within-tier* and
*between-tier* components. The **Theil T index** (GE(1)) supports an exact additive
decomposition:

$$T = T_{\text{between}} + T_{\text{within}} = \sum_g \frac{n_g}{N}\frac{\mu_g}{\mu}\ln\frac{\mu_g}{\mu} \;+\; \sum_g \frac{n_g}{N}\frac{\mu_g}{\mu} T_g$$
where $T_g$ is the within-group Theil T for group $g$, $n_g$ is the group count, $\mu_g$ is the
group mean capital, and $\mu$ is the overall mean.

```{r}
#| label: theil-decomp
#| fig-width: 9
#| fig-height: 4.5

theil_t <- function(x) {
  x  <- x[x > 0 & !is.na(x)]
  mu <- mean(x)
  mean((x / mu) * log(x / mu))
}

decomp <- co |>
  group_by(size) |>
  summarise(
    n_g = n(),
    mu_g = mean(capital_stock),
    T_g  = theil_t(capital_stock),
    .groups = "drop"
  ) |>
  mutate(
    N             = sum(n_g),
    mu            = sum(n_g * mu_g) / N,
    w_g           = (n_g / N) * (mu_g / mu),
    T_between_g   = w_g * log(mu_g / mu),
    T_within_g    = w_g * T_g
  )

T_between <- sum(decomp$T_between_g)
T_within  <- sum(decomp$T_within_g)
T_total   <- T_between + T_within

# Panel A: total within vs between
summary_bar <- tibble(
  component = c("Between tiers", "Within tiers"),
  value     = c(T_between, T_within),
  pct       = c(T_between / T_total, T_within / T_total)
)

p_summary <- ggplot(summary_bar, aes(x = component, y = pct, fill = component)) +
  geom_col(width = 0.55, alpha = 0.9) +
  geom_text(aes(label = percent(pct, accuracy = 0.1)),
            vjust = -0.5, fontface = "bold", size = 4.2) +
  scale_y_continuous(labels = percent_format(), limits = c(0, 1),
                     name = "Share of total Theil T") +
  scale_fill_manual(
    values = c("Between tiers" = "#264653", "Within tiers" = "#e76f51"),
    guide  = "none"
  ) +
  labs(title    = "Total decomposition",
       subtitle = glue("Total Theil T = {round(T_total, 3)}"),
       x = NULL) +
  theme_minimal(base_size = 12) +
  theme(plot.title = element_text(face = "bold", colour = "#264653"),
        panel.grid.major.x = element_blank(),
        panel.grid.minor   = element_blank())

# Panel B: within-tier contributions broken down by size
p_within <- decomp |>
  mutate(pct_of_total = T_within_g / T_total) |>
  ggplot(aes(x = fct_rev(size), y = pct_of_total, fill = size)) +
  geom_col(width = 0.55, alpha = 0.9) +
  geom_text(aes(label = percent(pct_of_total, accuracy = 0.1)),
            vjust = -0.5, fontface = "bold", size = 4.2) +
  scale_y_continuous(labels = percent_format(), limits = c(0, 0.7),
                     name = "Within-tier share of total Theil T") +
  scale_fill_manual(values = pal_size, guide = "none") +
  labs(
    title    = "Within-tier breakdown by size",
    subtitle = "Each bar = one tier's contribution to the within-group component",
    x = NULL
  ) +
  theme_minimal(base_size = 12) +
  theme(plot.title = element_text(face = "bold", colour = "#264653"),
        panel.grid.major.x = element_blank(),
        panel.grid.minor   = element_blank())

p_summary + p_within +
  plot_annotation(
    title   = "Most of Brazil's capital inequality is *within* size tiers, not between them",
    subtitle = glue(
      "Theil T decomposition. Within-tier component = {percent(T_within / T_total, 0.1)} ",
      "of total inequality — the largest driver is inequality inside micro-enterprises."
    ),
    caption = caption_base,
    theme   = theme(
      plot.title    = element_markdown(face = "bold", size = 14, colour = "#264653"),
      plot.subtitle = element_text(colour = "grey40", size = 10.5, lineheight = 1.3),
      plot.caption  = element_text(colour = "grey50", size = 8)
    )
  )
```

The decomposition is the key structural result of this section. If most inequality were
*between* tiers, then adjusting the revenue classification cutoffs could reduce aggregate
inequality. But if the dominant component is *within* tiers, revenue-threshold reform achieves
little: the same cutoff that separates micro from small cannot distinguish capital-poor
subsistence firms from well-endowed holding-company subsidiaries within the micro category.
The within-tier dominance directly motivates the governance analysis that follows.
